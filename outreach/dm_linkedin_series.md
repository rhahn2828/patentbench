## General Notes

- **Timing:** Send initial DMs Tuesday-Thursday, 9-11am recipient's time zone
- **Connection requests:** If not already connected, send a connection request with a brief note first. Do not send the full DM as a connection request message (too long, gets cut off)
- **Response tracking:** Log all outreach in [TRACKING SPREADSHEET/CRM] with date sent, response received, and outcome
- **Do not mass-send:** Personalize each message. LinkedIn flags and restricts accounts that send templated messages in bulk
